# Boss Fighters

A Python-based fighting game featuring multiple characters and boss battles.

## Version
v0.1.8

## Features
- Multiple playable characters (Archer, Axe, Knight, Druid, and more)
- Boss enemy encounters
- Character animations and sprites
- Combat system with attacks and projectiles

## Project Structure
- `bossfighters v*.py` - Different versions of the game
- `images/` - Game assets including character sprites, boss animations, and projectiles
  - `characters/` - Playable character sprites with attack and idle animations
  - `bosses/` - Boss enemy sprites
  - `player_projectiles/` - Player weapon projectiles
  - `boss_projectiles/` - Boss attack projectiles

## Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/bossfighters.git

# Install dependencies (if any)
pip install -r requirements.txt
```

## Usage
```bash
python "bossfighters v0.1.8.py"
```
